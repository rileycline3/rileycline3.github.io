    <h2>Unit 1 & 2 Vocabulary</h2>
    <div class="vocab-list">
      <p><strong>Bit:</strong> A binary digit, either 0 or 1.</p>
      <p><strong>Byte:</strong> A sequence of 8 bits.</p>
      <p><strong>Roundoff:</strong> Error from limited bits of precision.</p>
      <p><strong>Analog Data:</strong> Smoothly changing values, like sound.</p>
      <p><strong>Lossless:</strong> Compression that preserves all data.</p>
      <p><strong>Lossy:</strong> Compression that discards some data.</p>
      <p><strong>Metadata:</strong> Data about data.</p>
      <p><strong>Computing Device:</strong> A device that runs programs.</p>
      <p><strong>Computer Network:</strong> Interconnected computing devices.</p>
      <p><strong>Path:</strong> Series of network connections between sender and receiver.</p>
      <p><strong>Protocol:</strong> Rules for how systems communicate.</p>
      <p><strong>Fault Tolerant:</strong> Keeps working even if parts fail.</p>
      <p><strong>Scalability:</strong> Ability to grow with demand.</p>
      <p><strong>Bandwidth:</strong> Maximum transmission capacity.</p>
      <p><strong>URL:</strong> Web address, like www.example.com.</p>
      <p><strong>IP:</strong> Assigns addresses and routes data.</p>
      <p><strong>TCP:</strong> Reliable, error-checked delivery.</p>
      <p><strong>UDP:</strong> Faster, lightweight delivery.</p>
      <p><strong>Packets:</strong> Small chunks of data.</p>
      <p><strong>Routers:</strong> Forward data along the best path.</p>
      <p><strong>DNS:</strong> Translates domain names to IPs.</p>
      <p><strong>HTTP:</strong> Protocol for webpages.</p>
      <p><strong>WWW:</strong> System of linked pages and media.</p>
    </div>
